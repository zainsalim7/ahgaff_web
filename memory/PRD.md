# نظام الحضور - جامعة الأحقاف

## المتطلبات الأساسية
- نظام حضور مركزي لجامعة الأحقاف
- دعم أدوار متعددة: مدير، عميد، رئيس قسم، معلم، موظف، طالب
- 7 أنواع تقارير مع تصدير PDF/Excel
- استيراد بيانات الطلاب من Excel (عربي/إنجليزي)
- تطبيق موبايل (Expo/React Native) + واجهة ويب

## المهام المعلقة

### نظام التنبيهات الشامل (2026-02-28)
- **تنبيه تلقائي** عند إنشاء محاضرة جديدة → للمعلم + طلاب المقرر
- **إرسال يدوي** من الإدارة:
  - إلى جميع طلاب مقرر معين
  - إلى طالب بعينه (بحث)
  - إلى معلم بعينه (بحث)
  - للجميع أو حسب الدور
- APIs جديدة: `/notifications/my`, `/notifications/mark-read/{id}`, `/notifications/mark-all-read`, `/notifications/search-teachers`, `/notifications/courses`
- صفحة جديدة: `send-notification.tsx`
- حفظ الإشعارات داخل التطبيق + push notifications عبر Firebase
