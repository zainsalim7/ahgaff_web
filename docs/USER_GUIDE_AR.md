# نظام إدارة جامعة الأحقاف - دليل الميزات والاستخدام

## الهيكل التنظيمي
```
الجامعة ← الكليات ← الأقسام ← المقررات ← الطلاب
```

---

## 1. إدارة المستخدمين والصلاحيات (RBAC)

### الأدوار المتاحة:
| الدور | الوصف |
|-------|-------|
| admin | مشرف النظام - صلاحيات كاملة |
| dean | عميد كلية - يرى بيانات كليته فقط |
| registrar | مسجل - إدارة الطلاب والتسجيل |
| teacher | معلم - يرى مقرراته وطلابه |
| student | طالب - يرى جدوله وحضوره |
| employee | موظف - صلاحيات محدودة |

### الصلاحيات التفصيلية:
- إدارة المستخدمين (manage_users)
- إدارة الطلاب (manage_students)
- إدارة المعلمين (manage_teachers)
- إدارة المقررات (manage_courses, add_course, edit_course, delete_course)
- إدارة التسجيل (manage_enrollments)
- إدارة الأقسام (manage_departments)
- إدارة الكليات (manage_faculties)
- إدارة المحاضرات (manage_lectures, view_lectures)
- التقارير (view_reports, export_reports)
- الاستيراد (import_data)
- إدارة الإعدادات (manage_settings)

### طريقة الاستخدام:
1. اذهب إلى **الإدارة** > **المستخدمين**
2. أنشئ مستخدم جديد واختر الدور
3. لتخصيص الصلاحيات: اضغط على المستخدم > تعديل الصلاحيات
4. يمكن إنشاء **أدوار مخصصة** من الإدارة > الأدوار

---

## 2. إدارة الكليات

### الميزات:
- إضافة / تعديل / حذف كلية
- ربط الكلية بالأقسام
- إعدادات خاصة لكل كلية

### طريقة الاستخدام:
1. اذهب إلى **الإدارة** > **الإعدادات** > تبويب **الجامعة**
2. اضغط **+ إضافة كلية**
3. أدخل اسم الكلية ورمزها

---

## 3. إدارة الأقسام

### الميزات:
- إضافة / تعديل / حذف قسم
- ربط القسم بكلية
- عرض إحصائيات القسم (عدد الطلاب، المقررات، المعلمين)
- لوحة معلومات الأقسام

### طريقة الاستخدام:
1. اذهب إلى **الإدارة** > **الأقسام**
2. اضغط **+ إضافة قسم** واختر الكلية التابع لها

---

## 4. إدارة المقررات

### الميزات:
- إضافة / تعديل / حذف مقرر
- **إنشاء شعب متعددة تلقائياً** (أ، ب، ج...) حتى 10 شعب
- ربط المقرر بمعلم وقسم
- الخطة الدراسية لكل مقرر
- حذف آمن مع إمكانية الاستعادة من سلة المحذوفات

### طريقة الاستخدام:
1. اذهب إلى **المقررات** > **+ إضافة مقرر**
2. أدخل اسم المقرر، الرمز، عدد الساعات
3. اختر المعلم والقسم
4. حدد **عدد الشعب** إذا أردت إنشاء أكثر من شعبة

---

## 5. إدارة الطلاب

### الميزات:
- إضافة / تعديل / حذف طالب
- تفعيل / تعطيل حسابات الطلاب (فردي أو جماعي)
- استيراد الطلاب من ملف Excel
- تصدير بيانات الطلاب (Excel / PDF)
- إعادة تعيين كلمة المرور
- تغيير المستوى الدراسي بشكل جماعي
- حذف آمن مع إمكانية الاستعادة
- **QR Code** لكل طالب

### طريقة الاستخدام:
1. اذهب إلى **الطلاب** > **+ إضافة طالب**
2. أو اضغط **استيراد** لرفع ملف Excel
3. لتعطيل مجموعة: حدد الطلاب > **تعطيل جماعي**

---

## 6. إدارة المعلمين

### الميزات:
- إضافة / تعديل / حذف معلم
- استيراد المعلمين من ملف Excel
- تحديد النصاب الأسبوعي لكل معلم
- تفعيل / تعطيل حسابات المعلمين
- حذف آمن مع إمكانية الاستعادة
- إعادة تعيين كلمة المرور

### طريقة الاستخدام:
1. اذهب إلى **المعلمين** > **+ إضافة معلم**
2. أدخل البيانات وحدد **النصاب الأسبوعي** (الافتراضي 12 ساعة)

---

## 7. التسجيل في المقررات

### الميزات:
- تسجيل طالب في مقرر
- حذف تسجيل
- **نسخ جماعي** للطلاب بين المقررات
- **نقل جماعي** للطلاب بين المقررات
- استيراد تسجيلات من ملف Excel

### طريقة الاستخدام:
1. اذهب إلى **المقررات** > اختر مقرر > **الطلاب المسجلين**
2. اضغط **+ تسجيل طالب** أو **استيراد**
3. لنسخ/نقل: حدد الطلاب > **نسخ إلى مقرر** / **نقل إلى مقرر**

---

## 8. المحاضرات والجدول

### الميزات:
- **إضافة محاضرة** يدوياً
- **توليد محاضرات الفصل** تلقائياً (تحديد الأيام والأوقات لكل يوم)
- **كشف التعارضات** تلقائياً (منع تعيين معلم لمحاضرتين في نفس الوقت)
- إعادة جدولة المحاضرات (تاريخ مستقبلي فقط)
- إلغاء المحاضرات
- عرض الجدول الأسبوعي بتصميم Timeline
- عرض محاضرات اليوم
- عرض محاضرات الشهر
- **تصدير محاضرة PDF** مع تفاصيل الحضور

### طريقة الاستخدام:
#### توليد محاضرات الفصل:
1. اذهب إلى **المقررات** > اختر مقرر > **توليد**
2. حدد تاريخ البداية والنهاية
3. فعّل الأيام المطلوبة (السبت، الأحد، ...)
4. لكل يوم: اختر **وقت البداية** و **وقت النهاية** من القوائم المنسدلة
5. يمكن إضافة أكثر من محاضرة في اليوم الواحد
6. اضغط **توليد المحاضرات**

#### إعادة الجدولة:
1. اضغط على المحاضرة > **إعادة جدولة**
2. اختر التاريخ الجديد (مستقبلي فقط) والوقت
3. النظام يتحقق تلقائياً من عدم وجود تعارض

---

## 9. الحضور والغياب

### الميزات:
- تسجيل الحضور لمحاضرة (حاضر / غائب / متأخر / مستأذن)
- تسجيل حضور فردي سريع
- تعديل حالة الحضور بعد التسجيل
- عرض إحصائيات الحضور لكل طالب ومقرر
- مزامنة الحضور

### طريقة الاستخدام:
1. اذهب إلى المحاضرة > **تسجيل الحضور**
2. حدد حالة كل طالب
3. اضغط **حفظ**

---

## 10. التقارير

### التقارير المتاحة:
| التقرير | الوصف |
|---------|-------|
| ملخص عام | نظرة شاملة على النظام |
| نسب إتمام الدروس | كم تم إنجاز من الخطة الدراسية |
| نظرة عامة على الحضور | إحصائيات الحضور الشاملة |
| الطلاب الغائبين | قائمة بالطلاب كثيري الغياب |
| تقرير يومي | ملخص يوم محدد |
| تحذيرات الغياب | الطلاب الذين تجاوزوا حد الغياب |
| تقرير طالب | تفاصيل حضور طالب محدد |
| تقرير مقرر | تفاصيل مقرر محدد |
| تقرير قسم | إحصائيات القسم |
| ملخص المعلمين | أداء المعلمين |
| تأخيرات المعلمين | تقرير التأخيرات |
| نصاب المعلمين | الساعات المطلوبة مقابل المنفذة |

### حساب النصاب:
- **المطلوب** = النصاب الأسبوعي × عدد الأسابيع الكاملة
- **المنفذ** = ساعات المحاضرات التي سُجل فيها حضور فعلي
- **الفرق** = المنفذ - المطلوب (موجب = ساعات زائدة، سالب = نقص)

### التصدير:
- جميع التقارير قابلة للتصدير بصيغة **Excel** و **PDF**
- تقرير الفصل الدراسي الشامل (PDF)

### طريقة الاستخدام:
1. اذهب إلى **التقارير** من القائمة الرئيسية
2. اختر نوع التقرير
3. حدد الفترة الزمنية (اختياري)
4. اضغط **تصدير** للحصول على ملف Excel أو PDF

## 11. الجدول الأسبوعي

### الميزات:
- عرض جميع محاضرات الأسبوع بتصميم Timeline حديث
- الأيام: السبت إلى الجمعة
- تصفية حسب القسم أو المعلم
- عرض تفاصيل المحاضرة بالنقر عليها

### طريقة الاستخدام:
1. اضغط على تبويب **الجدول** في القائمة السفلية
2. استخدم الأسهم للتنقل بين الأسابيع
3. اضغط على أي محاضرة لعرض التفاصيل

---

## 12. الفصول الدراسية

### الميزات:
- إنشاء فصل دراسي جديد (خريف / ربيع / صيفي)
- تفعيل فصل دراسي (فصل واحد نشط فقط)
- إغلاق الفصل
- أرشفة الفصل
- إحصائيات الفصل

### طريقة الاستخدام:
1. اذهب إلى **الإعدادات** > تبويب **الفصول**
2. اضغط **+ إضافة فصل دراسي**
3. لتفعيل فصل: اضغط **تفعيل** بجانبه

---

## 13. الإعدادات العامة

### تشمل:
- **بيانات الجامعة**: الاسم، الرمز، الوصف، العنوان، الشعار
- **رفع شعار الجامعة**: من تبويب الجامعة > تعديل > رفع شعار
- **إعدادات الحضور**: حد التأخير، نسبة الغياب المسموحة
- **السنوات الأكاديمية**

---

## 14. أدوات الصيانة

### متاحة في: الإعدادات > أسفل الصفحة (للمشرف فقط)

| الأداة | الوظيفة |
|--------|---------|
| ربط الطلاب والمعلمين بالكليات | إصلاح بيانات الكلية المفقودة |
| ربط المقررات بالفصل الدراسي | ربط المقررات بالفصل النشط |
| إصلاح أدوار المستخدمين | إصلاح الأدوار المخصصة المعطلة |

> **استخدم هذه الأدوات مرة واحدة بعد كل تحديث كبير**

### الميزات:
- إشعارات تلقائية عند الأحداث المهمة
- إرسال إشعارات يدوية

## 17. سلة المحذوفات

### الميزات:
- عرض جميع العناصر المحذوفة (طلاب، معلمين، مقررات)
- استعادة عنصر محذوف
- حذف نهائي
- تفريغ السلة بالكامل

### طريقة الاستخدام:
1. اذهب إلى **الإدارة** > **سلة المحذوفات**
2. اضغط **استعادة** لإرجاع عنصر أو **حذف نهائي** لإزالته

---

## 18. سجل النشاطات

### الميزات:
- تسجيل تلقائي لجميع العمليات (إضافة، تعديل، حذف، تسجيل دخول)
- عرض من قام بالعملية ومتى
- إحصائيات النشاط

---

## 19. رفع الملفات والصور

### الميزات:
- رفع شعار الجامعة
- التخزين على **Emergent Object Storage** (سحابي خارجي)
- الأنواع المدعومة: JPEG, PNG, GIF, WebP
- الحد الأقصى: 5MB لكل صورة
- جاهز لإضافة صور الطلاب والمعلمين مستقبلاً
