# نظام حضور جامعة الأحقاف - PRD

## المشكلة الأصلية
نظام إدارة حضور الطلاب لجامعة الأحقاف

## البنية التقنية
- **Backend**: FastAPI + MongoDB | **Frontend**: Expo (React Native) | **Deploy**: Railway | **Notifications**: Firebase

## ما تم إنجازه

### الجلسات 1-5
- النظام الأساسي + حالة غياب تلقائي + PDF + صلاحيات + سجل تدقيق + مزامنة offline

### الجلسة 6 (2026-02-28)
1. إصلاح خلل تقرير الطالب (تصادم أسماء دوال get_user_permissions)
2. توحيد تحميل الصلاحيات (get_current_user يحمل من roles collection)
3. تجربة الطالب: تقرير شخصي تلقائي "تقرير حضوري" بدون فلاتر
4. تجربة المعلم:
   - بار سفلي: الرئيسية | المقررات | محاضراتي | حسابي
   - القائمة الجانبية: محاضراتي + مقرراتي + الجدول + تقارير محددة
   - نصاب التدريس: عنوان "نصاب تدريسي" - بياناته فقط - بدون فلاتر
   - تقرير الحضور: عنوان "تقرير حضور مقرراتي" - بدون فلتر أقسام
   - كل البيانات مفلترة: يرى فقط مقرراته وطلابه
5. إضافة صلاحية report_teacher_workload للمعلمين
6. إصلاح القائمة الجانبية لكل الأدوار
