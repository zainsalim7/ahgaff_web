# أستاذ الأحقاف - تطبيق الأستاذ
## Teacher App - Ahgaff University

تطبيق موبايل خفيف مخصص للأساتذة مع دعم تسجيل الحضور بدون إنترنت.

### الإعداد
```bash
yarn install
```

### تشغيل محلي
```bash
npx expo start
```

### بناء APK
```bash
npx eas build --profile preview --platform android
```

### رابط الـ Backend
الـ Backend مرفوع على Railway:
`https://ahgaffweb-production-c582.up.railway.app`

### الميزات
- تسجيل دخول الأستاذ
- لوحة القيادة مع المحاضرات والمقررات
- عرض المقررات مع إحصائيات الحضور
- تسجيل الحضور (يعمل بدون إنترنت!)
- مزامنة تلقائية عند عودة الاتصال
- الإشعارات
- الملف الشخصي وتغيير كلمة المرور
- قائمة طلاب كل مقرر
- سجل حضور المحاضرات
